enum Kunlaboro::MessageSystem::MessageLocality : uint8_t

The locality of the message.

This is used to differentiate between global and local message requests, as well as ones that should accept both global as well as local messages.

Enumerator
Message_Global 

The message is only accepted when sent globally.

Message_Local 

The message is only accepted when sent locally.

Message_Either 

Accept both global and local messages.
