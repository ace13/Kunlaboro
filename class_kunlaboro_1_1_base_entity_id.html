Detailed Description

template<typename idType, typename indexType, uint8_t IndexBits, typename generationType, uint8_t GenerationBits>
class Kunlaboro::BaseEntityId< idType, indexType, IndexBits, generationType, GenerationBits >

Defines an entity ID.

Entities are indexed based on two variables - to help uniqueness.

  • Index The index part is the position in memory where the entity is created.
  • Generation The generation part exists to allow reuse of small index values while still generating unique entity IDs.
Note
Depending on the generation width, entity IDs may become non-unique after reuse. If ID uniqueness is not an issue then generation width may be set to 1.
Todo:
Allow for 0-width generation values, when uniqueness is a non-issue.
Template Parameters
idType

The underlying ID type, recommended to be an unsigned integral.

(Guaranteed valid types are; uint8_t, uint16_t, uint32_t, uint64_t)

indexTypeA type wide enough to contain the index part of the ID.
IndexBitsThe width of the index part of the ID, must fit in the indexType.
generationTypeA type wide enough to contain the generation part of the ID.
GenerationBitsThe with of the generation part of the ID, must fit in the generationType.
Note
Total bit count of index and generation must match the number of bits in the underlying type.
